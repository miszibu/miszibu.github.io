Go中的接口

Go不是传统的面向对象的编程语言，它里面没有类和继承的概念。
但是Go语言中的接口概念可以很方便的实现许多面向对象的特性。

一个接口可以被多个类同时实现，接口具有多态性。
多态是OOP编程中的概念：同一个类型在不同的实例上表现出不同的行为。

类通过实现接口，来继承接口，注意如果在实现接口时，不加*，实际上是值传递，不能改变类的数据。

接口嵌套接口

一个接口可以包含一个或多个其他的接口。
