创建Dockerfile

1.我们需要打包一个muc项目的Image。因此需要muc的代码文件，如果直接拷贝Muc代码文件，需要将其的引用包也同步导入，不方便。直接使用go build 工具进行打包

2.在任意地方创建文件夹，在文件夹下放入Dockerfile。

为了方便和条理性，我在/var/lib/docker文件夹下创建，并且将相关文件移到该目录下
